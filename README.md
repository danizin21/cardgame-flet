CardGame – Flet (Python)

Um card visual inspirado em jogos, desenvolvido em Python utilizando a biblioteca Flet.
O projeto exibe informações de um personagem, incluindo imagem, descrição e atributos como dano, velocidade e defesa.

Funcionalidades

Interface gráfica moderna e responsiva

Exibição de personagem com imagem e descrição

Layout em formato de card

Seção de atributos (Dano, Velocidade e Defesa)

Design com gradiente, sombras e bordas arredondadas

Tecnologias utilizadas

Python 3

Flet

Pré-requisitos

Python 3 instalado

Biblioteca Flet instalada

Como executar

Certifique-se de ter o Python instalado

Instale a biblioteca necessária:

pip install flet


Execute o arquivo principal:
python main.py
